Дефолтовые значения
Значения которые передаются в бд для установки каких-то констант.
CREATE SEQUENCE serial 


Автоинкремент нужны для некоторых полей по умолчянию, что бы не вводить руками постоянно
CREATE TABLE cars (
    id SERIAL,
    prod_n integer,
    cost numeric DEFAULT 3.33
    );

SERIAL здесь значит,что при добавлении новой записи в таблицу id будет назначаться автоматически и не нужно будет указывать постоянно новый id.
DEFAUL принимает значения по умолчанию, к примеру если цена на многие товары одна и та же или значения колонок, что бы не вбивать руками каждый раз число мы задаём на определённое поле DEFAULT и при добавлении новой записи в таблицу ,если не указывать никакое значени ,будет присвоен DEFAULT в примере выше 3.33
